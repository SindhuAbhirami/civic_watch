# CivicWatch – AI-Based Civic Issue Image Classification

## Overview

CivicWatch is a web-based project where I used MobileNet to classify civic issues from images.

Instead of manually selecting a category, the user can upload an image, and the system will automatically predict the type of issue such as pothole, fire or trash.

This project mainly focuses on using AI along with basic web technologies like HTML, CSS, and JavaScript.


## Features

* Upload an image of a civic issue
* Automatic classification using MobileNet
* Lightweight and fast model
* Simple user interface
* Displays predicted result with confidence



## Model Details

* Model used: MobileNet (pre-trained CNN)
* Technique: Transfer Learning
* Framework: TensorFlow.js

### Working

* The user uploads an image
* The image is resized and processed
* It is passed to the MobileNet model
* The model predicts the category
* The result is displayed



## Tech Stack

Frontend:

* HTML
* CSS
* JavaScript

AI/ML:

* TensorFlow.js
* MobileNet



## How It Works

1. User uploads an image
2. Image is converted into tensor format
3. MobileNet processes the image
4. The model predicts the category
5. Result is shown on the screen



## Conclusion

CivicWatch shows how AI can be used with simple web technologies to identify civic issues from images.

The system is fast, easy to use, and can be improved further by adding more features in the future.
